# Join the HyperAgility Movement

HyperAgility is a new movement, it needs time to mature and it needs feedback from the ecosystem to become useful and sustainable, please help to redefine how we build software in the age of AI.

## How to Contribute

You can make contributions to the manifesto itself by submitting a PR or an Issue on Github.

- **GitHub:** [Submit a Pull Request or Issue](https://github.com/ciprian-cgr/hyperagility)
- **Discussion:** Join the conversation on our GitHub Discussions board.

If you feel there are other ways to contribute, please get in touch.
